OAuth credentials
DriveBackup uses your own Google Cloud project so your data goes directly to your Drive. These credentials are stored locally on your machine only.
Go to console.cloud.google.com → Create a new project (e.g. DriveBackup).
Enable the Google Drive API for your project under "APIs & Services → Library".
Go to APIs & Services → OAuth consent screen. Set app type to External, fill in an app name, and add your Gmail as a test user.
Go to Credentials → Create Credentials → OAuth client ID. Choose Desktop app. Copy the Client ID and Secret below.
Add http://localhost:42813/oauth2callback to the Authorized redirect URIs.
